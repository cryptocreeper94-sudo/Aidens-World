# The Chronoverse

**A free, ad-free, educational action-adventure game built for kids aged 6–9.**

Created by **Lume42 Labs**, **Academy.tlid.io**, and **DarkWave Studios LLC** as a philanthropic initiative.

🌐 **Play Now**: [aiden.tlid.io](https://aiden.tlid.io)

---

## What Is The Chronoverse?

The Chronoverse is a browser-based, offline-capable learning ecosystem where screen time actually builds cognitive skills. Inspired by Geometry Dash's addictive run-and-jump mechanics, it wraps reading comprehension, spatial logic puzzles, and digital safety concepts inside a superhero-themed adventure.

- **100% Free** — No ads, no micro-transactions, no algorithms tracking your child
- **Offline-Capable** — Works without an internet connection after first load
- **Safe Environment** — Operates on the Trust Layer infrastructure with zero external tracking
- **Parent-Child Co-Op** — Dedicated "Learn Mode" for collaborative discovery

## Features

### 🎮 Play Mode — Rift Runner
- Procedurally generated side-scrolling levels across 9 unique worlds
- 12 playable characters with unique abilities and projectile types
- Overdrive meter, Rift Portal dimension-hopping, and boss encounters
- Progressive difficulty with tower obstacles, enemies, and Chrono Shard collection

### 🧠 Learn Mode
- Parent + Child collaborative discovery activities
- Reading comprehension missions integrated into gameplay lore
- Digital safety and integrity concepts taught through gameplay

### 🔊 Create Mode — Lume Voice Sandbox
- Natural voice command interface for building custom logic rules
- Kids learn basic programming concepts through spoken instructions

### 🔓 Rift Locker
- Earn **Lumes ✦** (in-game currency) through achievements — never purchasable
- Unlock additional hero suits and villain skins
- 4 free base suits: Ironboy, Supergirl, Superboy, Cyborg Girl
- Aspirational unlockables: Spider Hero, Jedi Kid, Shadow Suit, Syndicate Thug, Imperial Trooper, Symbiote Brute

### 🃏 Refractor Cards
- Holographic character cards with animated foil effects
- Full stat breakdowns for each hero and villain
- Custom name input — kids can assign their own name to any character

## Tech Stack

- **Engine**: Phaser 3 (HTML5 Canvas)
- **UI**: Vanilla HTML/CSS/JS portal with glassmorphic design
- **Audio**: ElevenLabs AI narration for story panels
- **Hosting**: Static deployment via GitHub Pages / Vercel
- **Save System**: localStorage-based with full progression tracking
