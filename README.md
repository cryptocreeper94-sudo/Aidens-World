## 🎮 Game Modes

### ⚡ Rift Runner — Play Mode
The core action-adventure experience. Procedurally generated side-scrolling levels across **9 unique worlds**, each with distinct visual themes, obstacles, and enemy types.

- **12 playable characters** with unique abilities and projectile types
- **Overdrive meter** — fill it by collecting Chrono Shards to enter invincible burst mode
- **Rift Portal dimension-hopping** — mid-run dimensional shifts change the rules
- **Boss encounters** — epic battles at the end of each world
- **Progressive difficulty** — tower obstacles, enemy patterns, and environmental hazards scale up
- **Chrono Shard collection** — in-game currency earned through skill, never purchased

### 🧠 Learn Mode — Parent-Child Co-Op
A dedicated collaborative discovery environment designed for parent-child interaction:

- **Reading comprehension missions** woven into the game's lore and world-building
- **Digital safety concepts** taught through gameplay scenarios
- **Integrity and ethics modules** — characters model decision-making
- **No time pressure** — learn at your child's pace

### 🔊 Create Mode — Lume Voice Sandbox
A natural voice command interface where kids learn basic programming concepts through spoken instructions:

- "Make the hero jump higher" → modifies jump physics
- "Add more enemies" → spawns additional obstacles
- Kids learn cause-and-effect logic through play
- Powered by Lume language syntax under the hood
